Present continuous - Hiện tại tiếp diễn

1. Diễn tả 1 hành động đang xảy ra tại thời điểm nói.
For example: I'm talking with you.

2.Diễn tả 1 hành động đang xảy ra nhưng không nhất thiết diễn ra ngay lúc nói.
For example: I'm working for VNG Company.

3.Diễn tả những hành động có kế hoạch sắp xảy ra trong tương lai gần.
For example: I'm meeting you tonight.

4.Hành động thường xuyên lặp đi lặp lại gây sự bực mình hay khó chịu cho người nói.
For example: He is always losing his key (+) S + is/am/are + V-ing (-) S + is/am/are + not + V-ing (?) is/am/are + S + V-ing ? 1. Thông thường, chúng ta chỉ việc thêm đuôi "-ing" vào sau động từ.
For example: learn => learning, look => looking, work => working, sing => singing

2.Động từ tận cùng bằng "e", ta bỏ "e" rồi mới thêm "ing"
For example: take => taking, dance => dancing, make => making, ride => riding

3. Động từ tận cùng bằng "ee", chỉ việc thêm "ing"
For example: see => seeing, age => ageing, agree => agreeing, free => freeing

4. Động từ tận cùng bằng "ie", ta biến "ie" thành "y" rồi thêm "ing".
For example: lie => lying, die => dying (I'm dying = tao đang chết dần chết mòn)

Động từ tận cùng bằng "y" vẫn giữ nguyên khi thêm "ing"
For example: Carry => carrying, Stay => staying

* Quy tắc gấp đôi phụ âm cuối khi thêm "ing"
5. Với động từ 1 âm tiết, tận cùng bằng "Nguyên âm + Phụ âm" (Trừ h, w, x, y), ta gấp đôi phụ âm và thêm "ing"
For example: win => winning, stop => stopping

6. Với động từ 2 vần, trọng âm rơi vào âm tiết cuối, tận cùng bằng "nguyên âm + phụ âm", ta gấp đôi phụ âm và thêm "ing"
For example: Prefer(pre'fer) => preferring, begin(be'gin) => beginning

7. Với 3 động từ tận cùng là "C", bạn phải thêm chữ "K" ở cuối rồi mới thêm "ing"
For example: Traffic => trafficking, Mimic => mimicking, Panic => panicking. - At the moment, Now, Right now, At present. Đây là những dấu hiệu nhận biết của thì hiện tại tiếp diễn.
